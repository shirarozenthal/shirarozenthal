Project Description¶

On December 2nd, 2022, Pelosi sold an additional 20,000 shares of Visa, valued at ~$4M.

Back in 2008, Visa offered her husband a "VIP cut of the IPO" at $11/share

Since then, $V is up ~1900%

Just days after her sale of 20,000 shares, the Credit Card Competition Act, legislation aiming to break up the Visa-MA duopoly, was introduced to the House. If passed, Visa would no longer monopolize CC fees.

The Stock Trading on Congressional Knowledge Act requires U.S. Senators to publicly disclose all of their financial transaction within 45 days. In this project we analyze these disclosures, monitor trade performance in the time following transaction, and build a classification to determine, upon disclosure, whether a senator's trade will outperform the S&P500.

Project Goals¶

In this project, we hope to analyze the performance of senators' trades, look into any recurring patterns or trends in the dataset, and hopefully be able to classify whether a newly-disclosed trade will profitably outperform the market.

Senate Trade Disclosures¶

https://senatestockwatcher.com/api

This dataset contains a full list of nearly 10,000 senate trades from this past decade. Observations document the transaction and disclosure dates, senators, tickers, political party, sector, industry, and amount associated with every disclosed trade.

Observations document the following features associated with every disclosed trade:

  • Senator
  • Transaction Date
  • Disclosure Date
  • Stock Ticker
  • Political Party
  • Sector

Below, we insert an additional column, "days between", to quantify the number of days a senator waited before disclosing their trade (remember that the maximum is 45).

Webscraping Senator Committees¶

https://www.senate.gov/general/committee_assignments/assignments.htm

The above URL links to the senate.gov website where a list of the U.S. Senators and current committees they serve on can be found.

We chose to include this data because of the potential that the committees a senator belongs to could influence how they perform in certain industries or sectors. Using BeautifulSoup, this information is scraped into a pandas DataFrame.

The objective of this endeavor is to add dummy variable columns to the disclosures data for each committee.

Herding¶

The code below adds a herding feature to our disclosures DataFrame.

This feature is based on the number of senators that made the same decision (purchase v.s. sale) in the same stock over the past 50 days as a proportion of total transactions made in that stock over the same time period. The code also removes decisions made by the same senator from this calculation.

In [24]:
# Duration of herding interval
interval = 25
herding = []
# time_ is array of times so we can plot herding distribution over time
time_ = []

#iterate over every row in disclosures
for i in range(len(disclosures['type'])):
  ticker = disclosures['ticker'][i]
  type_ = disclosures['type'][i]
  
  # before is the disclosures that occurred before the date of disclosure in the interval
  before = disclosures[((disclosures['disclosure_date'] >= (disclosures['disclosure_date'][i] - timedelta(days=interval))) & (disclosures['disclosure_date'] <= (disclosures['disclosure_date'][i])))].reset_index(drop=True)
  
  # calculate the trades in the interval that were the same 'type' for the same 'ticker'
  agree = before[before['ticker']==ticker].reset_index(drop=True)
  agree = agree[agree['senator']!=disclosures['senator'][i]].reset_index(drop=True)
  if len(agree['senator'].unique()) != 0:
    if type_ == 'Purchase':
      herding.append((len(agree[agree['type']==type_]))/len(agree))
    else:
      herding.append((len(agree[agree['type']!='Purchase']))/len(agree))
  else:
    herding.append(0)
  # append time to time_
  time_.append(disclosures['transaction_date'][i])

# Insert new herding column
try:
  disclosures.insert(len(disclosures.columns), 'herding', herding)
except:
  disclosures.drop('herding', axis=1, inplace=True)
  disclosures.insert(len(disclosures.columns), 'herding', herding)

This code inserts another herding feature to our disclosures DataFrame.

This time, this feature is the ratio of number transactions coming from the same committee that were the same decision type in the same sector over 50 days to the total number of transactions made in that sector. We chose to include this feature because we wanted use the scraped committee data to see if transactions placed in high numbers in the same sector by people in the same committees result in higher returns

  for committee in unique_committees:
    if disclosures[committee][i] == 1:
      committees.append(committee)
  before = disclosures[((disclosures['transaction_date'] >= (disclosures['transaction_date'][i] - timedelta(days=interval))) & (disclosures['transaction_date'] <= (disclosures['transaction_date'][i])))].reset_index(drop=True)
  before = before[before['senator']!=disclosures['senator'][i]].reset_index(drop=True)
  before = before[before['industry']==disclosures['industry'][i]].reset_index(drop=True)
  num_total = len(before['senator'].unique())
  if num_total != 0:
    for a in range(len(before['type'])):
      typ = before['type'][a]
      tot = 0
      for committee in committees:
        if before[committee][a] == 1:
          if typedisclosure == 'Purchase':
            if typ == 'Purchase':
              tot+=1
              herding+=1
          else:
            if typ != 'Purchase':
              tot+=1
              herding-=1
        else:
          tot+=1

      herding = herding/tot

  else:
    herding = 0
  committee_herding.append(herding)
  time.append(disclosures['transaction_date'][i])
try:
  disclosures.insert(len(disclosures.columns), 'committee_herding', committee_herding)
except:
  disclosures.drop('committee_herding', axis=1, inplace=True)
  disclosures.insert(len(disclosures.columns), 'committee_herding', committee_herding)
